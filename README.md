# Superstore Sales Analysis (SQL + Tableau)

This project analyzes retail sales data to uncover trends in revenue and profitability.
SQL was used for data aggregation and analysis, and Tableau was used to build an interactive dashboard
highlighting sales seasonality, category performance, and regional profitability.

## Dataset
- Source: Superstore Sales dataset
- Records: ~9,900 orders
- Key fields: Order Date, Sales, Profit, Category, Region

## Tools Used
- SQL (SQLite): data cleaning, aggregation, and analysis
- Tableau Public: interactive data visualization and dashboard creation
- Excel: data inspection and validation

- Sales consistently peak in Q4 (Octoberâ€“December), indicating strong seasonal demand.
- Technology is the top-performing category in both sales and profit.
- Furniture shows relatively high sales but lower profit margins, suggesting pricing or cost issues.
- The West region leads in profitability, while the Central and South regions underperform.

## Recommendations
- Increase marketing and inventory focus during Q4 to capitalize on strong seasonal demand.
- Prioritize high-performing categories such as Technology to maximize revenue and profitability.
- Review pricing, discount strategies, and cost structure for Furniture to improve margins.
- Strengthen sales and marketing strategies in underperforming regions such as the South.
